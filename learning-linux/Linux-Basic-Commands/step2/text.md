Display messages in kernel ring buffer

```plain
dmesg
```{{exec}}

Display CPU information

```plain
cat /proc/cpuinfo
```{{exec}}

Display memory information

```plain
cat /proc/meminfo
```{{exec}}

Display free and used memory ( -h for human readable, -m for MB, -g for GB.)

```plain
free -h
```{{exec}}

Show info about disks

```plain
fdisk -l
```{{exec}}
